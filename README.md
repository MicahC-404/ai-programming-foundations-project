# AI Programming Foundations: Titanic Data Workflow

## Project Description
This project demonstrates a complete, reproducible data workflow using Python. I built a pipeline that ingests the Titanic dataset, cleans missing values, performs exploratory analysis, and visualizes key survival patterns to provide a foundation for future machine learning work.

## Dataset
- **Name:** Titanic - Machine Learning from Disaster
- **Source:** [Kaggle Titanic Dataset](https://www.kaggle.com/c/titanic/data)
